History of html

html is stands for Hyper Text Markup Language. It is a standard language which is use for creating a web page. HyperText Markup Language (HTML) is a markup language used to create web pages. It was created in 1990 by Sir Tim Berners-Lee,
he is a British computer scientist who is credited with inventing the World Wide Web. Webpages can include writing, links, pictures, and even sound and video. It is a markup language. and it is universaly accepted.
